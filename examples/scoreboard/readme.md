# Scoreboard

* Displays P1:P2 for a second
* Press button A to increment left 2 digits
* Press button B to increment right 2 digits

## Install

* Connect to micro:bit via USB
* Install v1.9.2 firmware

```
$ ufs put tm1637.py
$ ufs put examples/scoreboard/main.py
```
