# Counter

* Start with zero.
* Press button A to increment.
* Press button B to decrement.
* Number is changed and sent to the display on button up.

## Install

* Connect to micro:bit via USB
* Install v1.9.2 firmware

```
$ ufs put tm1637.py
$ ufs put examples/counter/main.py
```
