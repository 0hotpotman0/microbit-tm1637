# Gestures

* Listen for gestures: up, down, left, right, face up, face down, freefall, 3g, 6g, 8g, shake
* Show captured gesture

## Install

* Connect to micro:bit via USB
* Install v1.9.2 firmware

```
$ ufs put tm1637.py
$ ufs put examples/gestures/main.py
```
