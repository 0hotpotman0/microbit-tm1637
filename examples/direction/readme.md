# Direction

* Display compass heading on 4 digit display
* Display compass using clock hands on micro:bit led matrix

## Install

* Connect to micro:bit via USB
* Install v1.9.2 firmware

```
$ ufs put tm1637.py
$ ufs put examples/direction/main.py
```
