# LEDs

* Walk through all possible LED combinations
* Press any button to stop
* Press A to increment
* Press B to decrement

## Install

* Connect to micro:bit via USB
* Install v1.9.2 firmware

```
$ ufs put tm1637.py
$ ufs put examples/leds/main.py
```
